# Generated references (not committed)

Two files live here after you run the **setup agent**, both are gitignored because
they contain your personal data:

- `base_cv.md`, your canonical CV content: contact, experience, education, projects,
  and your three-tier skills sort. Every tailored CV is drawn from this file.
- `cv_design.md`, your **unique** python-docx design system, generated from
  `setup/knowledge/cv_design_axes.md` so no two JobPilot users share a CV look.

If they're missing, the cv-creation agent refuses to run. Fix: run the setup agent
(docs/SETUP.md step 10).
